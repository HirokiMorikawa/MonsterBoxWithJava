void controller.LoginController.doPost ( HttpServletRequest  request,
HttpServletResponse  response 
) throws ServletException, IOException
inlineprotected

ユーザ認証関係の処理をパス contextPath + /login に一任させる．クライアントから送られてきたHTTPリクエストメッセージが，POSTによるリクエストであったときに メッセージボディに格納されているリクエストパラメータを取得する． リクエストパラメータの変数名はuserid，password, f_typeと定義している． useridとpasswordには，ユーザ認証を行うためのユーザidとパスワードが格納されている． また，それとは別に，リクエストがどのような種類かを判断するためにf_typeという変数がある．これが扱う値はsubmit，sign_inもしくはcheckである． f_typeの値によってリクエストごとに行うべき処理を分離することができる
